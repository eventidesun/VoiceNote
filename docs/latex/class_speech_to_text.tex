\doxysubsection{Detailed Description}
A class for recording and transcribing audio. 

The \doxylink{class_speech_to_text}{Speech\+To\+Text} class provides the application with functionality to record audio, pause and stop recordings, track recording duration, and transcribe recorded audio using an external Whisper model.
